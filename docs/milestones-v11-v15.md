# MerlionOS v11-v15 里程碑规划

> 当前：v10.0.0 — 99 模块，20,000+ 行 Rust

---

## v11.0.0 — 真机联网 (Real Hardware Networking)

### 目标
在 HP 笔记本上通过 USB 以太网适配器连接互联网。

### 交付标准
- [ ] USB 以太网适配器驱动（CDC-ECM/ASIX）
- [ ] DHCP 自动获取真实 IP
- [ ] DNS 解析真实域名
- [ ] `ping 8.8.8.8` 成功
- [ ] `wget` 从真实网站下载页面

### 预计新增：~1500 行

---

## v12.0.0 — 图形界面 (GUI Foundation)

### 目标
基于帧缓冲实现简单的图形界面，支持窗口和鼠标。

### 交付标准
- [ ] 窗口管理器（可拖动的矩形窗口）
- [ ] 鼠标光标（PS/2 或 USB HID 鼠标）
- [ ] 图形化终端窗口（shell 在窗口内运行）
- [ ] 图形化系统监控（top 的图形版）
- [ ] 壁纸（纯色或简单图案）

### 预计新增：~2000 行

---

## v13.0.0 — 持久化与用户空间 (Persistence & Userland)

### 目标
独立的用户空间程序，持久化配置，多用户支持。

### 交付标准
- [ ] 从 NVMe/AHCI 磁盘启动（不只是 USB）
- [ ] ext2-like 持久化文件系统
- [ ] 独立的 merlion-user crate（交叉编译的用户程序）
- [ ] 用户程序从磁盘加载执行（ELF loader + 文件系统）
- [ ] `/etc/passwd` 多用户登录
- [ ] 密码哈希验证

### 预计新增：~2500 行

---

## v14.0.0 — AI 自主体 (Autonomous AI)

### 目标
AI 不再需要人工触发，能自主监控、修复、优化系统。

### 交付标准
- [ ] AI 通过网络直接调用 Claude API（HTTPS 或 HTTP 代理）
- [ ] Agent 自动调度器已就绪，定期运行 health/monitor
- [ ] AI 自动检测并杀死失控进程
- [ ] AI 根据负载自动调整调度优先级
- [ ] `ai journal` — AI 定期写系统日志摘要
- [ ] 自然语言系统管理：`ai 优化内存` → 自动执行

### 预计新增：~1500 行

---

## v15.0.0 — 生产就绪 (Production Ready)

### 目标
MerlionOS 可以作为一个轻量级服务器 OS 长期运行。

### 交付标准
- [ ] 稳定运行 24 小时无 panic
- [ ] SSH 服务器（或简化的远程 shell）
- [ ] 日志持久化到磁盘
- [ ] 看门狗自动重启
- [ ] 内存泄漏检测
- [ ] 完整的 man 手册覆盖所有命令
- [ ] 一键安装到硬盘（`install` 命令）
- [ ] 官网 merlionos.dev 展示

### 预计新增：~2000 行

- 一个**能在真机上长期运行**的操作系统
- 一个**有图形界面**的操作系统
- 一个**AI 自主管理**的操作系统
- 一个**近 30,000 行 Rust** 的正经开源项目
- 一个**完全由 AI 编写**的技术奇迹
